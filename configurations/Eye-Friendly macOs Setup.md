# Eye-Friendly macOS Coding Setup

## macOS

### Night Shift
- Warmth → `75%`
- Schedule → `Sunset to Sunrise`

### True Tone
- `ON`

---

# Lunar

https://lunar.fyi

## Enable
- Sync Mode
- Adaptive Brightness
- Smooth Transitions

## Settings
- Day Brightness Max → `45%`
- Night Brightness Max → `28%`
- Day Temp → `5200K`
- Night Temp → `3800K`

---

# BetterDisplay

https://betterdisplay.pro

## Enable
- HiDPI
- Native Smooth Scaling

## Use
- ONLY HiDPI resolutions
- Avoid blurry “Looks like …” scaling modes

---

# Monitor Hardware Settings

- Brightness → `22–35%`
- Contrast → `60–70%`
- Sharpness → `Low/Medium`
- Color Mode → `Warm` or `Reading`

---

# Terminal

- Font → `JetBrains Mono`
- Font Size → `16`
- Background → `#1E1E1E`
- Avoid pure black backgrounds

---

# Room Setup

- Warm room lighting
- No dark room coding
- Warm LED behind monitor (`2700K–3000K`)
- Monitor at arm-length distance
- Top of monitor slightly below eye level

---

# Most Important Rule

If eyes still hurt:
- LOWER brightness further

Brightness causes more fatigue than blue light.
